# c-string-lab
Experiment string in c

# Description
A small C string library implemented from scratch for practicing pointers, memory mgmt, and atandard C library behavior.

## Implemented functino
- my_strlen
- my_strcpy
- my_strncpy
- my_strcmp
- my_strncmp
- my_strchr
- my_strstr
- my_strdup
- my_strcat

## Build
```bash
make
```
